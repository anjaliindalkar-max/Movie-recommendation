# 🎬 Movie Recommendation System

A simple and beginner-friendly **Movie Recommendation System** built using **Python** and **Tkinter**. This application recommends movies based on the genre of the movie selected by the user.

## 📌 Features

* Simple and clean graphical user interface (GUI)
* Select a movie from the dropdown menu
* Displays movie details such as:

  * Genre
  * Release Year
  * IMDb Rating
* Recommends similar movies based on the selected movie's genre
* Recommendations are sorted by rating

## 🛠️ Technologies Used

* Python 3
* Tkinter (GUI Library)

```bash

```

2. Move into the project folder.

```bash
cd movie-recommendation-system
```

3. Run the Python file.

```bash
python movie_recommendation.py
```

## 📖 How It Works

1. Open the application.
2. Select a movie from the dropdown list.
3. Click the **Get Recommendations** button.
4. The application displays:

   * Selected movie details
   * Genre
   * Rating
   * Release year
   * A list of recommended movies from the same genre

## 🎥 Available Genres

* Sci-Fi
* Fantasy
* Romance
* Action

Each genre contains multiple movies, and recommendations are generated by matching the selected movie's genre.
